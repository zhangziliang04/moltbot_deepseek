# `moltbot pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:
- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
moltbot pairing list whatsapp
moltbot pairing approve whatsapp <code> --notify
```
